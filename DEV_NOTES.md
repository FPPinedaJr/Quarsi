# DEV_NOTES

## EXTENSIONS
- Headwind (ryan heybourn)
- PHP (devsense)
- Tailwind CSS IntelliSense (tailwind labs)
- Tailwind Fold (stivo)

## TAILWIND
- Install Tailwind
```bash
npm install
```

- Create Tailwind input.css File
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```


- Link Tailwind output.css File
```html
<link rel="stylesheet" href="./css/output.css">
```
- Run Tailwind
```bash
npm run dev
npm run build:admin
npm run build:officer
npm run build:student
npm run build:sign_in
npm run build:global
```
## JQUERY
- Include Jquery JS File
```html
<script src="../assets/js/jquery-3.7.1.min.js"></script>
```

## FONT AWESOME
- Link Fontawesome CSS Files
```html
<link rel="stylesheet" href="../assets/css/fontawesome/all.min.css"> 
<link rel="stylesheet" href="../assets/css/fontawesome/fontawesome.min.css">
```

## FAVICON
- Link favicon image
```html
<link rel="icon" href="./assets/images/favicon.ico" type="image/x-icon">
```

## GLOBAL CSS
- Link global output.css
```html
<link rel="stylesheet" href="../assets/css/output.css?v=1.0">
```
